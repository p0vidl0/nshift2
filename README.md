# Контроллер задержки активации второго канала

## Работа
Время задержки настраивается резистором R5 от 0 до 255 мс.
При помощи перемычки SV1 можно увеличить время задержки в два раза.

## Спецификация
1. ATtiny13A-SSU
2. CNY17-1 DIP
3. LM7805
4. IRF510
5. CA9V
6. Немного резисторов и емкостей 1206
7. Электролит 100uF
